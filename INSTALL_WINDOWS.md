# Installation Guide for Windows

This document provides detailed instructions for installing and using the Ascend UPSC application on Windows.

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- 4GB RAM (minimum)
- 500MB free disk space
- 1280x720 screen resolution (minimum)

## Installation Options

You can choose between two installation methods:

### Option 1: Portable Executable (Recommended)

This option allows you to run the application without installation.

1. Download the `AscendUPSC-Portable.exe` file from the provided link
2. Save the file to any location on your computer
3. Double-click on the executable file to run the application
4. The application will start immediately with a pre-loaded database

**Benefits of portable mode:**
- No installation required
- Can be moved between computers easily (e.g., using a USB drive)
- Doesn't modify your system registry or create system files
- All data is self-contained within the application folder

### Option 2: Standard Installation

This option installs the application like a standard Windows program.

1. Download the `AscendUPSC-Setup.exe` installer from the provided link
2. Run the installer and follow the on-screen instructions
3. Choose an installation location when prompted
4. Wait for the installation to complete
5. Launch the application using the desktop shortcut or Start menu entry

## First-Time Setup

When running the application for the first time:

1. The application will start automatically and initialize its database
2. You'll see the home page with available tests
3. No login or registration is required
4. You can start using the application immediately

## Using the Application

### Taking Tests

1. From the home page, select a test you want to attempt
2. Click "Start Test" to begin
3. Answer the questions within the allotted time
4. Review your answers before submitting
5. View your test results and analytics

### Viewing Analytics

1. After completing a test, you can view detailed analytics
2. Navigate to the "Overall Analytics" section to see your performance across all tests
3. Review subject-wise performance, time spent, and accuracy metrics
4. Use the insights to focus on weaker areas

### Using Flashcards

1. Navigate to the "Flashcards" section
2. Review flashcards created from your previous incorrect answers
3. Rate your understanding of each concept
4. The spaced repetition system will schedule reviews based on your ratings

## Troubleshooting

If you encounter any issues:

### Application Doesn't Start

1. Ensure you have the necessary system requirements
2. Try running the application as administrator
3. Make sure your antivirus software isn't blocking the application
4. For portable version, try moving it to a different folder

### Database Issues

1. The application might show an error if the database is corrupted
2. For portable version, try downloading a fresh copy of the application
3. For installed version, try reinstalling the application

### Performance Issues

1. Close other applications to free up system resources
2. Restart the application if it becomes slow after extended use
3. Ensure your computer meets the minimum system requirements

## Data Storage

- For portable version: All data is stored within the application directory
- For installed version: Data is stored in the user's AppData folder

## Updates

When a new version is available:
- For portable version: Download the new portable executable
- For installed version: Run the new installer (your data will be preserved)

## Uninstallation

- For portable version: Simply delete the executable file
- For installed version: Use Windows Control Panel > Programs and Features > Uninstall
