# Portable Usage Guide for Ascend UPSC

This document provides detailed instructions on how to effectively use the portable version of Ascend UPSC.

## Getting Started with the Portable Version

The portable version of Ascend UPSC can be run directly without installation. This is perfect for using the application on multiple computers or carrying it on a USB drive.

### Running the Application

1. Double-click on the `AscendUPSC-Portable.exe` file to start the application
2. Wait for the application to load (this may take a few seconds)
3. The application will automatically open in your default web browser
4. You can immediately start using all features without any setup or login

### Data Location

All your data is stored within the portable application directory. This means:
- Your test attempts
- Performance analytics
- Flashcards
- Application settings

All will be preserved within the application's folder structure, making it easy to backup or transfer.

## Key Features

### Test Taking Interface

The application provides a comprehensive test-taking experience:

1. **Starting a Test**
   - From the home screen, browse available tests
   - Click on a test to view details
   - Click "Start Test" to begin an attempt
   - Answer questions within the allotted time
   - Submit when complete

2. **Test Review**
   - After submitting, review your answers
   - See correct answers for questions you missed
   - View detailed explanations
   - Analyze your performance with visual analytics

### Analytics Dashboard

The analytics dashboard provides comprehensive insights into your performance:

1. **Test-specific Analytics**
   - Access by clicking "View Analytics" after completing a test
   - See time spent on each question
   - View accuracy by subject and topic
   - Identify knowledge gaps

2. **Overall Analytics**
   - Navigate to "Overall Analytics" from the main menu
   - View performance trends across all tests
   - See subject-wise strengths and weaknesses
   - Track progress over time with visual charts

### Flashcard System

Convert your learning into effective revision materials:

1. **Creating Flashcards**
   - Flashcards are automatically created from questions you answer incorrectly
   - You can also manually create flashcards from any question

2. **Reviewing Flashcards**
   - Navigate to the "Flashcards" section
   - Cards are organized by subject and difficulty
   - Rate your understanding to help the spaced repetition system
   - Focus on challenging concepts based on your performance

## Tips for Optimal Usage

### Performance Optimization

To ensure the best experience with the portable version:

1. **Launch Location**
   - Run the application from a local drive rather than a network drive for better performance
   - If using a USB drive, ensure it's USB 3.0 or faster for optimal speed

2. **Resource Usage**
   - Close unnecessary applications when running Ascend UPSC
   - The application works best with at least 4GB of available RAM

### Data Backup

Since all data is stored within the application folder:

1. **Regular Backups**
   - Make periodic copies of the entire application folder to preserve your data
   - This is especially important if you're using the application on a USB drive

2. **Safe Ejection**
   - Always properly eject USB drives after using the application
   - Close the application completely before removing any storage device

## Troubleshooting Common Issues

### Application Won't Start

If the application fails to launch:

1. Check that you have administrator privileges on the computer
2. Ensure your antivirus isn't blocking the application
3. Try running the executable as an administrator (right-click > Run as administrator)
4. Make sure all required files are present in the application folder

### Database Errors

If you encounter database-related error messages:

1. Ensure the application wasn't interrupted during a previous session
2. Check that your storage device has sufficient free space
3. If errors persist, try using a backup copy of the application

### Browser Integration Issues

If the application launches but the browser interface doesn't appear:

1. Check that you have a default web browser set on your system
2. Try manually navigating to `http://localhost:3000` in your browser
3. Restart the application and try again
