# Ascend UPSC User Guide

Welcome to Ascend UPSC - your comprehensive preparation platform for UPSC exams!

## Installation Guide for Windows

### One-Click Installation

1. **Download the Installer**
   - Download the `AscendUPSC-Setup.exe` file from the provided link
   - Save it to a temporary location on your computer

2. **Run the Installer**
   - Double-click the downloaded `.exe` file
   - If a security prompt appears, click "Yes" to allow the installation
   - The installer will automatically:
     - Install the application to your Program Files directory
     - Create a desktop shortcut
     - Add an entry to your Start menu
     - Set up the database with initial content

3. **Launch the Application**
   - After installation completes, the application will launch automatically
   - For future use, you can start the application from:
     - The desktop shortcut
     - The Start menu
     - The installed location

### Alternative: Portable Version

If you prefer not to install the application on your system, we also offer a portable version:

1. **Download the Portable Executable**
   - Download `AscendUPSC-Portable.exe` from the provided link
   - Save it to any location on your computer or external drive

2. **Run Directly**
   - Double-click the executable to run the application
   - No installation is required
   - All data will be stored in the same folder as the executable

## Getting Started

### First Launch

When you first launch Ascend UPSC, you'll see the home page showing available practice tests.

### Taking a Test

1. **Select a Test**
   - Browse through the available tests on the home page
   - Click on a test to view its details

2. **Start the Test**
   - Click the "Start Test" button
   - The timer will begin automatically

3. **Answer Questions**
   - Read each question carefully
   - Select your answer from the options provided
   - Navigate between questions using the Next/Previous buttons
   - Use the question navigator to jump to specific questions

4. **Submit Your Test**
   - When you're finished, click "Submit Test"
   - You'll see your results immediately

### Reviewing Test Results

After completing a test, you can review:
- Your overall score
- Time spent on each question
- Correct and incorrect answers
- Detailed explanations for all questions

### Using Flashcards

The flashcard feature helps you memorize important concepts:

1. Click on "Flashcards" in the navigation menu
2. Browse through available flashcards
3. Use the "Flip" button to see answers/explanations
4. Mark cards as "Known", "Reviewing", or "Difficult"

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- 4GB RAM (minimum)
- 500MB free disk space
- 1280x720 screen resolution (minimum)

## Troubleshooting

### Application Won't Start

If the application doesn't start:
1. Ensure you have administrator privileges
2. Check that your antivirus isn't blocking the application
3. Try reinstalling the application

### Performance Issues

If the application is running slowly:
1. Close other applications to free up system resources
2. Restart the application
3. Ensure your computer meets the minimum system requirements

## Data Management

### Backing Up Your Data

For the installed version, your data is stored in:
```
C:\Users\[YourUsername]\AppData\Roaming\ascend-upsc\db
```

For the portable version, data is stored in the application folder.

To back up your data, create a copy of the database file.

### Uninstalling

To uninstall the application:
1. Open Windows Control Panel
2. Go to "Programs and Features" or "Apps & features"
3. Find "Ascend UPSC" and select "Uninstall"
